32.最长有效括号

sankigan2026-7-6算法LeetCode

最长有效括号open in new window

题目描述

给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

示例

输入：s = "(()"
输出：2
解释：最长有效括号子串是 "()"

输入：s = ")()())"
输出：4
解释：最长有效括号子串是 "()()"

输入：s = ""
输出：0

解答

栈

关键思路

利用栈来记录下标，通过 当前下标 - 栈顶下标 来计算有效括号子串的长度。

核心技巧：栈底始终保留一个「最后一个未匹配的右括号的下标」（初始为 -1），作为长度计算的基准位。这样每次匹配成功后，就能直接用当前位置减去这个基准位得到有效长度。

具体步骤

  1. 初始化栈，压入 -1 作为哨兵（作为初始基准位）
  2. 遍历字符串：
    • 遇到 (：将其下标压入栈
    • 遇到 )：先弹出栈顶（尝试匹配一个 (）
      • 如果栈空了，说明当前 ) 没有匹配的 (，将当前下标压入栈作为新的基准位
      • 如果栈非空，说明匹配成功，用 i - 栈顶下标 更新最大长度
  3. 返回最大长度

function longestValidParentheses(s: string): number {
  // 栈底放 -1 作为基准位（哨兵）
  const stack: number[] = [-1];
  let maxlen = 0;

  for (let i = 0; i < s.length; ++i) {
    if (s[i] === '(') {
      // 左括号直接入栈
      stack.push(i);
    } else {
      // 右括号先弹栈尝试匹配
      stack.pop();
      if (!stack.length) {
        // 栈空说明当前 ) 无法匹配，将其作为新的基准位
        stack.push(i);
      } else {
        // 匹配成功，用当前下标减去栈顶（基准位）得到有效长度
        maxlen = Math.max(maxlen, i - stack[stack.length - 1]);
      }
    }
  }

  return maxlen;
};

复杂度分析

  • 时间复杂度：O(n)，字符串遍历一次
  • 空间复杂度：O(n)，栈最坏情况下存储所有字符

动态规划

关键思路

定义 dp[i] 表示以下标 i 结尾的最长有效括号子串的长度。

有效括号子串必然以 ) 结尾，所以只需在 s[i] === ')' 时考虑状态转移。

具体步骤

对于 s[i] === ')' 的情况，分两种子情况讨论：

情况一：s[i - 1] === '('，即形如 ...()

  • 直接与前一个字符配对，dp[i] = dp[i - 2] + 2

情况二：s[i - 1] === ')'，即形如 ...))

  • 需要跨过 dp[i - 1] 长度的有效子串，去看它前面的字符
  • 设 j = i - dp[i - 1] - 1，如果 s[j] === '('，则可以与 s[i] 配对
  • 此时 dp[i] = dp[i - 1] + 2 + dp[j - 1]
    • dp[i - 1]：中间一段已经匹配好的有效长度
    • + 2：本次新匹配的 s[j] 和 s[i] 这一对
    • dp[j - 1]：s[j] 前面可能还接着另一段有效括号

代码中把两种情况合并了：由于当 s[i - 1] === '(' 时 dp[i - 1] === 0，此时 j = i - 1，恰好落到情况一，所以一个公式即可覆盖。

function longestValidParentheses(s: string): number {
  // dp[i] 表示以 s[i] 结尾的最长有效括号长度
  const dp = new Array(s.length).fill(0);
  let maxlen = 0;

  for (let i = 1; i < s.length; ++i) {
    // 只有以 ) 结尾才有可能构成有效括号
    if (s[i] === ')') {
      // 跨过前面已匹配的子串，找到可能与 s[i] 配对的位置
      const j = i - dp[i - 1] - 1;
      if (j >= 0 && s[j] === '(') {
        // 中间已匹配长度 + 本次新匹配的 2 + s[j] 之前的匹配长度
        dp[i] = dp[i - 1] + 2 + (j - 1 >= 0 ? dp[j - 1] : 0);
        maxlen = Math.max(maxlen, dp[i]);
      }
    }
  }

  return maxlen;
}

复杂度分析

  • 时间复杂度：O(n)，只需一次遍历
  • 空间复杂度：O(n)，dp 数组的存储开销
