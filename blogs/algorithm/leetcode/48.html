48.旋转图像

sankigan2026-7-1算法LeetCode

旋转图像open in new window

题目描述

给定一个 n × n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在 原地 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要 使用另一个矩阵来旋转图像。

示例

输入：matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出：[[7,4,1],[8,5,2],[9,6,3]]

输入：matrix = [[5,1,9,11],[2,4,8,10],[13,3,6,7],[15,14,12,16]]
输出：[[15,13,2,5],[14,3,4,1],[12,6,8,9],[16,7,10,11]]

解答

转置 + 水平镜像

直接按"旋转"的定义去搬运元素，需要处理 4 个位置构成的循环交换，边界和索引容易写错。更简洁的做法是把旋转拆成两个基本操作的复合：先转置，再对每一行做水平镜像。两个操作都原地可做，代码极其简洁。

下面用坐标映射严格证明两者的等价性。

为什么「转置 + 水平镜像」=「顺时针旋转 90°」

记矩阵阶数为 n，元素的行列下标为 (i, j)，规定 行向下增长、列向右增长。

定义三种操作对任意元素 (i, j) 的作用：

操作映射
顺时针旋转 90°(i, j) -> (j, n-1-i)
转置（沿主对角线翻转）(i, j) -> (j, i)
水平镜像（每行左右翻转）(i, j) -> (i, n-1-j)

① 先推旋转 90° 的映射

顺时针转 90° 后，原位置 (i, j) 的元素落到新位置 (j, n-1-i)。两个坐标分量分别看：

  • 新行坐标 = j：原列索引变成新行索引。直观上，原第 j 列的元素整体移动到了新矩阵的第 j 行。
  • 新列坐标 = n-1-i：原行索引变成新列索引，且做了翻转（原最上行 i=0 落到新最右列 n-1，原最下行 i=n-1 落到新最左列 0）。

所以 (i, j) -> (j, n-1-i)。

② 再算「转置 → 水平镜像」的复合映射

把复合拆成两步，把第一步的输出当作第二步的输入：

转置:          (i, j) -> (j, i)
水平镜像:      (row, col) -> (row, n-1-col)

复合: 先转置得 (j, i)，再代入水平镜像
      => (j, n-1-i)

两个映射 (i, j) -> (j, n-1-i) 完全一致，所以对任意元素，先转置再水平镜像的最终落点与顺时针旋转 90° 相同，整个操作等价。

3×3 实例对照

用 matrix = [[1,2,3],[4,5,6],[7,8,9]] 验证一遍，跟踪元素 1 和 3 的坐标：

原始             转置后            水平镜像后 (= 旋转 90°)
1 2 3            1 4 7            7 4 1
4 5 6     →      2 5 8      →     8 5 2
7 8 9            3 6 9            9 6 3

1: (0,0) → 转置 → (0,0) → 镜像 → (0,2)   顺时针 90°: (0,0) → (0,2) ✓
3: (0,2) → 转置 → (2,0) → 镜像 → (2,2)   顺时针 90°: (0,2) → (2,2) ✓

实现要点

  • 转置：沿主对角线交换 matrix[i][j] 与 matrix[j][i]，只需遍历主对角线右上方区域（j > i），否则会交换两次还原。
  • 水平镜像：对每一行 reverse()，即把 matrix[i][j] 与 matrix[i][n-1-j] 对调。

两个操作均为原地修改，无额外矩阵分配。

复杂度

  • 时间：O(n²)，转置遍历 n²/2 对，镜像遍历 n²/2 个元素，合计仍是 O(n²)。
  • 空间：O(1)，仅使用常数个临时变量，原地修改。
Code
/**
 Do not return anything, modify matrix in-place instead.
 */
function rotate(matrix: number[][]): void {
  const n = matrix.length;

  // 1. 转置: 沿主对角线翻转, 只遍历右上三角 (j > i), 避免交换两次还原
  for (let i = 0; i < n - 1; ++i) {
    for (let j = i + 1; j < n; ++j) {
      [matrix[i][j], matrix[j][i]] = [matrix[j][i], matrix[i][j]];
    }
  }

  // 2. 水平镜像: 每行左右翻转
  matrix.forEach((row) => row.reverse());
}
