912.排序数组

sankigan2025-2-25算法LeetCode

排序数组open in new window

题目描述

给你一个整数数组 nums，请你将该数组升序排列。

你必须在 不使用任何内置函数 的情况下解决问题，时间复杂度为 O(nlog(n))，并且空间复杂度尽可能小。

示例

输入：nums = [5,2,3,1]
输出：[1,2,3,5]

输入：nums = [5,1,1,2,0,0]
输出：[0,0,1,1,2,5]

解答

快速排序

  1. 选择第一个元素作为"基准"（pivot）
  2. 遍历剩余元素，小于 pivot 的放入 left，大于等于 pivot 的放入 right
  3. 递归对 left、right 分别排序，再拼接为 [...left, pivot, ...right]

这种写法直观易懂，但每次递归都会创建新数组，空间复杂度较高（O(n)），且以固定首元素作为 pivot，在有序/近有序数组上会退化为 O(n²)。

Code
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var sortArray = function(nums) {
  if (nums.length < 2) return nums;
  const left = [], right = [];
  const pivot = nums[0];
  for (let i = 1; i < nums.length; ++i) {
    if (nums[i] < pivot) {
      left.push(nums[i]);
    } else {
      right.push(nums[i]);
    }
  }
  return [...sortArray(left), pivot, ...sortArray(right)];
};

快速排序（三路分区 + 原地排序）

上一种解法每次递归都新建数组，空间开销大；这里改用原地分区，并引入三路分区（Dutch National Flag）应对大量重复元素的场景，同时随机选取 pivot 避免有序数组导致的最坏情况。

核心思路：

  1. 随机选取 pivot：在 [lk, rk] 范围内随机选一个索引作为 pivot，交换到最左侧，避免固定选首元素在有序数组上退化为 O(n²)
  2. 三路分区：维护 pivotstart、pivotend、scan 三个指针，一次遍历将区间划分为三部分：
    • [lk, pivotstart) — 小于 pivot 的元素
    • [pivotstart, pivotend] — 等于 pivot 的元素（无需再排序）
    • (pivotend, rk] — 大于 pivot 的元素
  3. 递归排序：分区后，等于 pivot 的中间区间已经就位，只需对左右两个子区间递归调用 quickSort

时间复杂度：平均 O(nlogn)，重复元素多时优于普通二路分区；空间复杂度：O(logn)（递归栈），原地交换无需额外数组

Code
function sortArray(nums: number[]): number[] {
  /**
   * 三路分区：将 nums[lk..rk] 划分为 [小于pivot | 等于pivot | 大于pivot]
   * @returns 等于 pivot 区间的起止索引 [pivotstart, pivotend]
   */
  function partition(nums: number[], lk: number, rk: number): [number, number] {
    // 随机选取 pivot 索引，将其交换到最左侧，避免有序数组的最坏情况
    const randomIdx = lk + Math.floor(Math.random() * (rk - lk + 1));
    const pivot = nums[randomIdx];
    [nums[randomIdx], nums[lk]] = [nums[lk], nums[randomIdx]];

    let pivotstart = lk;  // 等于区间的左边界
    let pivotend = rk;    // 等于区间的右边界
    let scan = lk + 1;    // 当前扫描指针

    while (scan <= pivotend) {
      if (nums[scan] < pivot) {
        // 小于 pivot：交换到左侧（小于区），pivotstart 和 scan 均右移
        [nums[pivotstart], nums[scan]] = [nums[scan], nums[pivotstart]];
        ++pivotstart;
        ++scan;
      } else if (nums[scan] > pivot) {
        // 大于 pivot：交换到右侧（大于区），pivotend 左移
        // scan 不前进，因为交换过来的元素尚未检查
        [nums[pivotend], nums[scan]] = [nums[scan], nums[pivotend]];
        --pivotend;
      } else {
        // 等于 pivot：无需交换，scan 直接跳过
        ++scan;
      }
    }

    return [pivotstart, pivotend];
  }

  /**
   * 对 nums[lk..rk] 进行原地快速排序
   */
  function quickSort(nums: number[], lk: number, rk: number): void {
    if (lk >= rk) return;

    const [pivotstart, pivotend] = partition(nums, lk, rk);
    // 等于 pivot 的区间已就位，只需递归排序左右两侧
    quickSort(nums, lk, pivotstart - 1);
    quickSort(nums, pivotend + 1, rk);
  }

  quickSort(nums, 0, nums.length - 1);

  return nums;
}
