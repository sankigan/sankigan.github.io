151.反转字符串中的单词

sankigan2026-6-29算法LeetCode

反转字符串中的单词open in new window

题目描述

给你一个字符串 s ，请你反转字符串中单词的顺序。

单词是由非空格字符组成的字符串。s 中使用至少一个空格将字符串中的单词分隔开。

返回单词顺序颠倒且单词之间用单个空格连接的结果字符串。

注意：输入字符串 s 中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

示例

输入：s = "the sky is blue"
输出："blue is sky the"

输入：s = "  hello world  "
输出："world hello"
解释：反转后的字符串中不能存在前导空格和尾随空格。

输入：s = "a good   example"
输出："example good a"
解释：如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。

提示

  • 1 <= s.length <= 10^4
  • s 包含英文大小写字母、数字和空格 ' '
  • s 中至少存在一个单词

进阶：如果字符串在你使用的编程语言中是一种可变数据类型，请你使用 O(1) 额外空间复杂度的原地解法。

解答

关键思路

题目本质是两件事：反转单词顺序 + 规范化空格（去掉首尾、折叠中间多余空格）。

直接 从右向左扫描，遇到单词整段提取并顺序压入结果数组，空格一律跳过——两件事一趟扫描同时完成：

  1. 游标 scan 从末尾往前走，遇到空格直接跳过（无论多少个，自然折叠）；
  2. 遇到非空格，启动内层循环逐字符拼出完整单词后 push 进 res；
  3. 最终 res.join(' ') 连接即可。

内层循环从右往左读，每个新字符拼到已得片段的前面（seg = s[scan--] + seg），保证单词内部字符顺序不变。

复杂度

  • 时间：O(n)，每个字符被访问一次。
  • 空间：O(n)，结果数组与单词片段。

进阶 O(1) 空间：先整体反转字符串，再对每个单词单独反转，需要字符串可变。

Code
function reverseWords(s: string): string {
  const res: string[] = [];
  let scan = s.length - 1;

  while (scan >= 0) {
    // 跳过空格段，定位到下一个单词的末尾
    if (/\s/.test(s[scan])) {
      scan--;
      continue;
    }

    // 收集一个完整单词：从右往左读，所以新字符拼在已得片段的前面
    let seg = '';
    while (scan >= 0 && !/\s/.test(s[scan])) {
      seg = s[scan--] + seg;
    }
    res.push(seg);
  }

  return res.join(' ');
}
